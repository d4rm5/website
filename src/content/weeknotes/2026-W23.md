## Como ser un 1000x Stremio user

Renové la suscripción a [Real Debrid](https://real-debrid.com), que es básicamente un sistema de cache de torrents. Si tenés una suscripción, podés agregar tu API Key al addon de Torrentio. Cuando vas a ver un contenido Torrentio consulta si RD ya tiene ese torrent cacheado en sus servidores. Si sí (indicado como `[RD+]`), reproducís un stream HTTP directo desde la infraestructura de RD a velocidad plena.

Ahí te evitas:

- Seeders lentos o escasos
- Problemas con tu ISP
- IP expuesta en el swarm
- Fuentes poco confiables
